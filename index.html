Selenide. Concise UI Tests in Java.

Selenide is a library for writing concise, readable, boilerplate-free tests in Java using Selenium WebDriver.

@Test
public void testLogin() {
  open("/login");
  setValue(By.name("user.name"), "johny");
  $("#submit").click();
  waitUntil("#username", hasText("Hello, Johny!"));

  $("#insuranceDetailsHeader").shouldHave(text("Страховые полисы"));
  assertEquals(7, $$("#paymentScheduleTable tr"));
}

How to start?

Just put selenide.jar to your project. If you are using Maven (Ivy, Gradle or whatever dependency management), you can add the following dependency: 
dependencies {
  testCompile 'com.codeborne:selenide:1.8'
}
Include the following methods and let's start! 

include static com.codeborne.selenide.Navigation.*
include static com.codeborne.selenide.DOM.*
You can find some Selenide usages in the example project Hangman.
