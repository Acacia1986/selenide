Selenide

Concise UI Tests in Java

@Test
public void userCanLoginByUsername() {
  open("/login");
  $(By.name("user.name")).setValue("johny");
  $("#submit").click();
  waitUntil("#username", hasText("Hello, Johny!"));

  $("#insuranceDetailsHeader").shouldHave( text("Страховые полисы") );
  assertEquals(7, $$("#paymentScheduleTable tr").size());
}

As you can see, Selenide provides:

  • jQuery-like element selectors
  • Ajax support
  • Automatic handling of browser lifecycle

How to start?

Just put selenide.jar to your project.
For Maven, Ivy, Gradle etc., users: 
<dependency org="com.codeborne" name="selenide" revision="1.8"/>
Import the following methods and let's start! 
import static com.codeborne.selenide.Navigation.*
import static com.codeborne.selenide.DOM.*
You can find some Selenide usages in the example project Hangman.
